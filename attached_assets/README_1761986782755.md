
# Shorol Notes (সরল নোট) — Bangla-first Notes with Voice → Transcript → Summary → Calendar

A starter Expo (React Native) app focused on Bangla note-taking. Record audio, transcribe (stub), summarize (stub), detect meeting times in Bangla, and push to Google/Outlook calendars (stubs). Offline-first with SQLite.

## Quickstart (Windows-friendly)
1. Install Node.js LTS and `npm i -g expo` (or `npx expo`).
2. In Cursor/VS Code: open this folder, run:
   ```bash
   npm i
   npm run start
   ```
3. Use Expo Go on your phone to scan the QR and try it immediately.

## Features
- ✅ Record Bangla voice (Expo AV)
- ✅ Notes list + search, edit title/body
- ✅ Save audio file attached to a note
- ✅ One-click **Transcribe + Summarize** (demo stubs)
- ✅ Detect Bangla date phrases and create calendar events (demo stubs)
- ✅ Local SQLite storage; offline-first
- ✅ Settings screen to store API keys/tokens
- ✅ Tabs: Notes | Calendar | Settings

## Wiring real providers
> For production, move API calls server-side.

- **Transcription (Whisper)**: put your key in Settings. Replace stub in `lib/speech.js` with a secure API route that calls OpenAI Whisper or Google STT for `bn`.
- **Summarization**: replace `summarizeBn()` in `lib/summarize.js` with your LLM endpoint.
- **Google Calendar**: implement OAuth with `expo-auth-session` and call Google Calendar API, then return `{ id }` and save via `linkEvent()`.
- **Outlook**: same with Microsoft Graph Calendar.

## Data model
- `notes(id, title, body, isImportant, summary_tldr, transcript_text, createdAt, updatedAt)`
- `media(id, noteId, type, uri, durationMs)`
- `events(id, noteId, provider, providerEventId, title, startsAt, endsAt)`
- `config(k,v)`

## Common tasks
- Add new text note: Notes tab → “টেক্সট”
- Record new voice note: Notes tab → “রেকর্ড”
- Transcribe + summarize: open a note → button “ট্রান্সক্রাইব + সামারি”
- Create Google/Outlook event: note page → choose provider

## Caveats
- OAuth/API code paths are **stubs** — you must plug secrets via a secure backend.
- Date parser is **naive**; extend `utils/dateParserBn.js` for richer Bangla variants.
